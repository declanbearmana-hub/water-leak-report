## Overview

The Water Leak Detection Investigation Report is an offline-first HTML application developed specifically for Archers Underground Services.

The application provides a professional digital workflow for recording, managing and exporting water leak detection investigations while producing clean, consistent client reports.

Designed for use on laptops, tablets and mobile devices, the application remains fully functional without an internet connection.
